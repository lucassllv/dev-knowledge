JavaScript: Lógica e interatividade

O JavaScript é o cérebro da página. É uma linguagem de programação completa que roda no navegador, permitindo criar interatividade real: validação de formulários, mapas interativos, atualizações de conteúdo sem recarregar a página e animações complexas. Ele transforma documentos estáticos em aplicações web dinâmicas.

Comando Descrição Exemplo
var / let / const Cria variáveis para guardar dados let nome = "Lucas";
function Cria um bloco de ação reutilizável function somar() {...}
console.log() Mostra mensagens no inspetor (F12) console.log("Oi");
if / else Toma decisões (Se / Senão) if (idade > 18) {...}
document.getElementById() Busca um elemento HTML pelo ID (o comando mais usado) let titulo = document.getElementById("meuTitulo");
.innerHTML Lê ou altera o conteúdo HTML interno de um elemento elemento.innerHTML = "Novo Texto";
.style Acessa e altera o CSS do elemento direto pelo JS elemento.style.color = "blue";
.value Pega o valor digitado em um campo de formulário let nome = campoInput.value;
onclick Executa algo quando o elemento é clicado <button onclick="minhaFuncao()">
addEventListener() Jeito profissional de "escutar" eventos (clique, tecla, mouse) botao.addEventListener("click", somar);
[ ] (Array) Cria uma lista de dados let frutas = ["Maçã", "Banana"];
.push() Adiciona um item novo no final da lista frutas.push("Laranja");
.length Conta quantos itens tem na lista (ou letras num texto) frutas.length; // Retorna 3
