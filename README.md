# dev-knowledge
O DevKnowledge nasce da necessidade de centralizar o fundamentos da web em uma interface moderna, intuitiva e livre de distrações. Enquanto a informação na internet é fragmentada, este projeto propõe uma utilidade direta: servir como um "canivete suíço" de consulta rápida para devs iniciantes sobre sintaxe e semântica de HTML, CSS e JavaScript.
