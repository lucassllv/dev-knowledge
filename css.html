CSS: Estilo e layout

O CSS (Folhas de Estilo em Cascata) é a camada de apresentação da web. Ele tem o poder de transformar o esqueleto bruto do HTML em interfaces visualmente atraentes, gerenciando cores, tipografia, espaçamentos e layouts complexos. É a tecnologia chave para garantir a responsividade (adaptação para múltiplos dispositivos).

Propriedade Descrição Exemplo
color Muda a cor do texto color: red;
background-color Muda a cor de fundo background-color: #000;
font-size Tamanho da fonte font-size: 16px;
margin Espaço externo (fora da borda) margin: 20px;
padding Espaço interno (entre o conteúdo e a borda) padding: 15px;
border Desenha uma linha ao redor do elemento border: 1px solid black;
width / height Define largura e altura width: 100%;
display: flex Transforma o elemento em um container flexível display: flex;
justify-content Alinha itens na horizontal (ex: centro, espaçado) justify-content: center;
align-items Alinha itens na vertical align-items: center;
flex-direction Define se a lista é em linha ou coluna flex-direction: column;
text-align Alinhamento do texto (esquerda, centro, direita) text-align: center;
font-weight Peso da fonte (negrito) font-weight: bold;
border-radius Arredonda os cantos das bordas border-radius: 10px;
cursor Muda o ícone do mouse (ex: mãozinha) cursor: pointer;
